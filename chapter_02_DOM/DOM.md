# DOM

### (Document Object Model)

&nbsp;

**👩🏻‍🏫 이번 파트에서 공부할 것들**

- 사람이 만든 웹페이지나 웹앱을 Browser가 어떻게 분석하여 정확한 위치에 표시하는지
- 우리(사람)가 어떻게 DOM 요소를 조작할 수 있는지
- (심화) Browser가 렌더링하는 순서
- 어떤 식으로 CSS를 쓰고 어떤 animation을 써야 Browser 성능에 좋은지
